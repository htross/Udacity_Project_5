To start the application:

1) Click on index.html which should open your default web browser.

2) You should see text fields for the address and city, a scrollable selection field for the state,
   and the submit button.

To run the application:

1) Enter a address, city, and state and hit submit

2) After all the data is populated for that address, you can filter/search
    - Search operates both on restaurant names (e.g., Tony's Pizza) or types (e.g., Pizza)
    - Distance sets the maximum distance (in feet) you are willing to travel from your present location.
